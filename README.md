# MCP MySQL Schema Inspector

A TypeScript-based MCP server for inspecting MySQL schemas.

## Features

- Uses `@modelcontextprotocol/sdk` for MCP integration
- Connects to MySQL databases using `mysql2`
- Loads configuration with `dotenv`
- TypeScript-first development

## Project Structure

- `src/index.ts` – Entry point
- `src/mysql-inspector.ts` – MySQL schema inspection logic (to be implemented)
- `src/config.ts` – Configuration loader

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Configure environment variables in a `.env` file.
